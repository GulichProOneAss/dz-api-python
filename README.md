## Задание 2 — OpenWeatherMap
Файл: `task2_openweather.py`  
Интерактивная программа: запрашивает название города и показывает текущую температуру и описание погоды (на русском языке).

**Важно:**  
Для работы второго скрипта требуется действующий API-ключ OpenWeatherMap. Замените строку  
`API_KEY = "1e6d1b880c148e0ef1e5e36f759a4cab"`  
на свой ключ (получить можно бесплатно на https://home.openweathermap.org/api_keys).

Проверено на Python 3.13.
